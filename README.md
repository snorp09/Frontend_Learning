# Front End Learning Repo
I'm using this repo to track my progress in re-familiarizing myself with Front End Developement, via React and NextJS.

This is the first major time I'll be working trying to use the NextJS App Router.

The end goal of this project is a simple Task Listing program, with (hopefully) minimal auth, and DB program for the backend, and Shadcn for the front end styling.

## Phases:

These are the goals I have for this project, and an explaination of what each part folder contains.

- ### Part 1: Minimal Prototype. 
No styling, no backend, storing the tasks using local storage. (Despite this, I'm targetting this phase to easily modified once the backend part is reached.)

- ### Part 2: Styling. 
This is where I'll use shadcn to try and make this look like a real finished project. Since this reaches my primary goal of familiarizing myself with the front end stack, this phase is my primary goal.

- ### Part 3: Backend & Users. 
Once I reach this phase, time permitting, I will add a backend using NextJS and Prism ORM. (Probably some form of Postgres as the DB.) This is my stretch goal, since my primary goal is not to learn the NextJS backend stack, but simply the front end stack.

If Part 3 is completed, I will create a new repository for that phase of the project independently, though I will also include it here.
This repo should be linked as part of that final repository, so I can show off my progress and my learning/adapting style and mentality.
